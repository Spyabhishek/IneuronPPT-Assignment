Q.2 What are the Different Types of Selectors in CSS & what are the advantages of them?

Ans :  Different types of Selectors are :-

1. Element Selector : The element selector selects HTML elements based on the element name or tag name. Ex - <p>, <h1>, etc

2. ID selector : This id selector uses the id attribute of a HTML element to select a specific element. Ex - #id_name

3. Class selector : The class selector selects HTML elements with a specific class attribute. Ex - .class_name

4. Universal selector : The Universal selector (*) in CSS is used to select all the elements in an HTML document. It also includes other elements which are inside under another element. Ex - * {}

5. Group selector : This selector is used to style all comma-separated elements with the same style. Ex - #div_name, h1, .class_name{

} 

6. Attribute selector : The attribute selector is used to select the elements with a specified attribute or attribute value. Ex : [href] {};

7. Pseudo-Class Selector : It is used to style a special type of state of any element. Ex : h1:hover{};

8. Pseudo-Element selector : It is used to style any specific part of the element. Ex : p::first-line{};
